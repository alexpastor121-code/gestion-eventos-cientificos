# Aplicación de gestión de eventos científicos
Esta aplicación está diseñada para facilitar la organización de congresos, simposios y charlas científicas, permitiendo la gestión de ponentes, horarios y registros de asistentes de forma eficiente.

##Funcionalidades previstas
*Registro y gestión de ponencias científicas.
*Gneración automática de certificados de asistencia.
*Calendario interactivo con el cronograma del evento.
